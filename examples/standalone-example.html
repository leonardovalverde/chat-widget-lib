ðŸ“‹ Integration Guide

Getting started is incredibly simple. Just add these two lines to your HTML: 

<!-- 1. Configure the widget -->
<script>
  window.ChatWidgetConfig = {
    apiKey: "your-openai-api-key",
    branding: {
      botName: "Your Assistant",
      colors: {
        primary: "#3b82f6",
        userMessageBg: "linear-gradient(135deg, #3b82f6, #8b5cf6)"
      }
    }
  };
</script>

<!-- 2. Load the widget -->
<script src="widget.umd.js"></script>

That's it! Your AI-powered chat widget is now live and ready to engage with your users.
